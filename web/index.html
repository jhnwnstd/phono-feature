Phono Feature is a browser tool for distinctive feature phonology. Select any set of speech sounds and it shows the natural class they form, the features they share, the features that distinguish them, and the minimal feature bundle that uniquely characterizes the set. You can also work in reverse and list every segment that matches a plus or minus feature specification. It runs entirely in your browser via Pyodide (Python compiled to WebAssembly), so nothing is uploaded to a server.

Features

  • Natural class analysis: the shared features, the distinguishing features, and the minimal distinctive bundle for any segment set.
  • Query by feature: find every phoneme that matches a plus or minus feature specification.
  • Inventory editor: build and edit IPA segment inventories, or load them from the PHOIBLE 2.0 database.
  • Runs fully in the browser with no install and no server. Ships Hayes (2009) features and a general IPA inventory.

FAQ

What is a natural class?
A set of speech sounds that share one or more distinctive features to the exclusion of every other sound in the inventory. Phono Feature computes it for any selection.
Does my data leave the browser?
No. The engine is Python compiled to WebAssembly and runs locally; inventories you load or build stay on your device.
Which feature systems does it support?
It ships Hayes (2009) features and a general IPA inventory, can load any inventory from PHOIBLE 2.0, and lets you build your own.
