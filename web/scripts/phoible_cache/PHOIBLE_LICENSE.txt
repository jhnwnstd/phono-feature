Per-language citation
---------------------

PHOIBLE inventories are aggregated from multiple primary sources
(SPA, UPSID, PHOIBLE, Eurasian Phonologies, etc.); when citing a
specific language's inventory, the PHOIBLE README directs users to
also credit the original source documented in PHOIBLE's
``InventoryID`` / ``Source`` fields.

What we include
---------------

We bundle a pre-processed snapshot of ``phoible.csv``:

  * an index of language names + inventory descriptors
    (``_phoible_index.generated.json``), and
  * the per-inventory segment-feature bundles
    (``_phoible_data.generated.json``).

Both are derived from PHOIBLE 2.0's raw CSV. The raw CSV itself
ships gzipped under ``web/scripts/phoible_cache/phoible.csv.gz`` so
the derivative artifacts can be rebuilt verbatim at any time.

No other files in this repository are derived from PHOIBLE.
